1. 문제 내의 변수.
- 명령어 조합 갯수 : int t
- 층 : int k (1 <= k) 
- 호 : int n ( n <= 14)

2. 구현
- 0층은 1, 2, 3 ... n (최대 14)이 됨
- 1층은 1, 1+2, 1+2+3 ... 1부터 n까지의 더한 수가 됨.
- 2층은 1, 1+(1+2), 1+(1+2)+(1+2+3)이 됨 
- 다시 말해서 한 층의 사람들의 정보를 저장해두면 다음 층의 인원을 쉽게 구할 수 있음.
