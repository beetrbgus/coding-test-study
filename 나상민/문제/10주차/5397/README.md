### 문제 접근 방법
에디터 문제와 같이 두 개의 스택을 가지고 문제를 풀었음.
처음에 시간초과가 나서 출력문을 bufferwriter로 수정하고, for문을 each문으로 수정하여 시간을 단축함.
### 구현 배경 지식
연결리스트, 스택

### 문제를 해결한 방법
1. 두 개의 스택을 만든다음 문자별로 case문을 태운다.
2. `<`인 경우에는 오른쪽 스택에 추가
3. `>`인 경우에는 왼쪽 스택에 추가
4. `-`인 경우에는 왼쪽 스택에서 제거
5. 나머지의 경우에는 왼쪽 스택에 추가
6. 왼쪽 스택을 오른쪽 스택에 다 추가 한 다음 출력
